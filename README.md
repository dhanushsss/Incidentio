# Effivio

**Kubernetes & Cloud Cost Optimization Platform** for IT and consulting organizations. FinOps-focused on cost, efficiency, and infrastructure clarity.

This repository contains the marketing/landing website for Effivio. Founded by Dhanush S.

## What Effivio Does

- **Visibility & cost attribution** — Per-pod and per-namespace Kubernetes cost; AWS Cost Explorer sync; unified K8s + AWS view; cloud accounts page.
- **Cost analytics & reporting** — Cost by cluster/namespace/deployment; executive dashboard; tenant-branded monthly PDF reports with audit.
- **Optimization & waste reduction** — Rule-based recommendations (right-sizing, idle/zombie, unused PVC, replica optimization); optimization dashboard with potential savings.
- **Budget & alerts** — Monthly budget limits; 70%/90% alerts; email and Slack; statistical anomaly detection.
- **Multi-tenancy & security** — SUPER_ADMIN, TENANT_ADMIN, TENANT_VIEWER; tenant isolation; full audit trail; JWT auth; encrypted secrets; token-authenticated ingestion.
